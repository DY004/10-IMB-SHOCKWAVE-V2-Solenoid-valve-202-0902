【*】 程序简介 
-工程名称：基于freertos的智能语音小冰箱
-实验平台: STM32 F407ZET6

【 ！】功能简介：
1-所有的程序都进行包装，放在while里面循环-轮询模式

2-轮询加中断执行模式

3-轮询加中断+定时器中断模式-执行模式

【 ！】实验操作：

通过按下KEY1以设置PID的目标值,按下KEY2以使能\失能PID计算.

打开野火多功能试助手----PID调试助手,在调试助手中打开开发板对应串口,上电开发板,按下KEY1\KEY2即可观察PID调整过程.

【 ！！】注意事项：
1-液位加水的重启的问题，得到解决，把放液位状态的存放位置，往后移动一点点位置，就可以实现相关功能了。

/*********************************************************************************************/

【*】 引脚分配

将开发板串口与电脑连接.
	
/*********************************************************************************************/

【*】 时钟

A.晶振：
-外部高速晶振：25MHz
-RTC晶振：32.768KHz

B.各总线运行时钟：
-系统时钟 = SYCCLK = AHB1 = 168MHz
-APB2 = 84MHz 
-APB1 = 42MHz

C.浮点运算单元：
  不使用
